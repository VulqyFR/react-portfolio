# **Portfolio**

Ce projet est un portfolio personnel développé en utilisant React avec Vite, offrant une configuration minimale pour faciliter le développement et la mise en place rapide.

## **Fonctionnalités**
- **React avec Vite :** Utilisation de Vite pour la création rapide de projets React avec un temps de démarrage ultra-rapide et des rechargements à chaud (HMR) pour un flux de travail de développement fluide.
- **ESLint Configuration :** Configuration préétablie avec certaines règles ESLint pour maintenir la qualité du code et garantir les bonnes pratiques de développement.
- **Plugins Officiels :** Ce projet utilise deux plugins officiels pour Vite :
    - **`@vitejs/plugin-react`**: Utilise Babel pour Fast Refresh (rafraîchissement rapide) pour des mises à jour instantanées lors de la modification du code.
    - **`@vitejs/plugin-react-swc`**: Utilise SWC pour Fast Refresh, offrant également des mises à jour instantanées sans rechargement de la page.


## **Installation**
```
git clone <lien-du-repo>
```
**Installation des Dépendances :** Accédez au répertoire du projet et installez les dépendances nécessaires en utilisant npm ou yarn :

```
npm install
# ou
yarn install
```

## **Utilisation**
**Développement :** Lancez le serveur de développement avec la commande :

```
npm run dev
# ou
yarn dev
```
Cela lancera l'application en mode développement. Ouvrez votre navigateur à l'adresse indiquée dans la console pour voir votre portfolio.

**Build :** Pour créer une version optimisée pour la production, utilisez la commande :

```
npm run build
# ou
yarn build
```
Cela générera les fichiers prêts à être déployés dans le dossier dist.
